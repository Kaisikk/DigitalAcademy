По итогу чтобы запустить мой jar просто закиньте его в директорию где лежат ваши файлы для тестов, и подайте в качестве аргументов только имя файла.
Пример java -jar task3.jar tests.json values.json
Сам файл report.json сформируется в той же директории, где был запущен файл, если файл в этой директории уже существует, то упадет ошибка
